Crea una aplicación en React con un formulario de registro que incluya validación en el cliente (JavaScript) y que se conecte con un servidor backend Node.js + Express.

## 1. Estructura del Formulario

**FormularioValidacion (Componente Principal):**

- Maneja el estado del formulario usando `useState` con la siguiente estructura:
  ```javascript
  const [formData, setFormData] = useState({
    nombre: "",
    email: "",
    password: "",
    confirmarPassword: "",
    moduloFavorito: "",
  });
  const [errors, setErrors] = useState({});
  ```

**Campos requeridos:**

- **Nombre**: Campo de texto
- **Email**: Campo de email
- **Contraseña**: Campo de password
- **Confirmar Contraseña**: Campo de password
- **Módulo Favorito**: Select con opciones: Seleccione un módulo, DWCS, DWCC, Interfaces, Despregamento, Inglés

## 2. Validaciones en el Cliente (JavaScript)

Implementa una función `validarFormulario()` que retorne un objeto con los errores encontrados:

**Nombre:**

- Obligatorio
- Mínimo 3 caracteres (después de trim)

**Email:**

- Obligatorio
- Formato válido: usar regex `/^[^\s@]+@[^\s@]+\.[^\s@]+$/`

**Contraseña:**

- Obligatoria
- Mínimo 6 caracteres

**Confirmar Contraseña:**

- Obligatoria
- Debe coincidir exactamente con el campo Contraseña

**Módulo Favorito:**

- Debe seleccionar una opción válida

## 3. Comportamiento del Formulario

**Al escribir en un campo:**

- Limpiar el error de ese campo específico mientras el usuario escribe
- Actualizar el estado `formData` con el nuevo valor

**Al enviar (submit):**

1. Ejecutar validación completa
2. Si hay errores: mostrarlos bajo cada campo correspondiente
3. Si no hay errores: enviar datos al servidor mediante `fetch`

**Endpoint del servidor:**

```javascript
POST http://localhost:3000/api/registro
Headers: { "Content-Type": "application/json" }
Body: JSON.stringify(formData)
```

**Manejo de respuestas:**

- Si el servidor responde OK (201): mostrar mensaje de éxito y limpiar el formulario
- Si el servidor responde con errores (400): mostrar los errores del servidor
- Si hay error de conexión: mostrar mensaje de error de conexión

## 4. Funcionalidades Adicionales

**Botón Limpiar:**

- Resetea todos los campos del formulario
- Limpia todos los errores

**Atributo noValidate:**

- Agregar al `<form>` para desactivar la validación HTML5 nativa

## 5. Ayuda

**Estructura básica handleChange:**

```javascript
const handleChange = (e) => {

  setFormData({ ...formData, [name]: value });
```
